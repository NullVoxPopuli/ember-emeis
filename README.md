The frontend for the [emeis](https://github.com/projectcaluma/emeis) user management service

## Compatibility

- Ember latest LTS (3.16)

## Installation

```bash
$ ember install ember-emeis
```

Then add the following lines to your `app/styles/app.scss`:

```scss
@import "ember-uikit";
@import "ember-emeis";
```
